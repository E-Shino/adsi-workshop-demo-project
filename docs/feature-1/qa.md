# 打刻メモ入力機能 — 要件 Q&A

## 対象タイミング

[Question] メモを入力できるタイミングはいつですか？（出勤時のみ / 退勤時のみ / 両方 / 打刻後に後から追加・編集も可能）
[Answer]両方

## 入力の必須/任意

[Question] メモの入力は必須ですか？それとも任意（空欄で打刻可能）ですか？
[Answer]任意

## 文字数制限

[Question] メモの最大文字数はどのくらいですか？（例: 50文字 / 200文字 / 制限なし）
[Answer]200文字

## 入力形式

[Question] メモは自由テキストのみですか？それともカテゴリ選択（例: 「直行」「直帰」「在宅」）＋自由テキストの組み合わせですか？
[Answer]カテゴリ選択＋自由テキスト形式

## 編集・削除

[Question] 一度入力したメモを後から編集・削除できますか？できる場合、期限はありますか？（例: 当日中のみ / いつでも可 / 管理者のみ編集可）
[Answer]いつでも可

## 閲覧範囲

[Question] メモは誰が閲覧できますか？（本人のみ / 上長も閲覧可 / 管理者全員が閲覧可）
[Answer]上長も閲覧可能

## 表示箇所

[Question] メモはどの画面に表示しますか？（打刻直後の確認表示 / 勤怠履歴一覧 / 日次詳細 / すべて）
[Answer]すべて

## UI の配置

[Question] メモ入力欄は打刻ボタンの近くに常時表示しますか？それとも打刻ボタン押下後にモーダル/ダイアログで入力させますか？
[Answer]打刻ボタン押下後に表示

## 追加質問（回答を踏まえて）

### カテゴリの選択肢

[Question] カテゴリの選択肢は何ですか？具体的に列挙してください。（例: 直行 / 直帰 / 在宅 / 外出 / 早退 / その他）また、カテゴリは複数選択可能ですか？
[Answer]具体例：直行、直帰、電車遅延、早退、外出、障害対応、その他　カテゴリは1つのみ選択可能

### カテゴリの必須/任意

[Question] カテゴリ選択は必須ですか？（メモを書く場合はカテゴリも必ず選ぶ / カテゴリなしで自由テキストのみも可）
[Answer]カテゴリ選択は必須

### メモの紐付き単位

[Question] メモは打刻レコード（出勤・退勤それぞれ）に1つずつ紐づきますか？それとも1日の勤怠に対して1つですか？（例: 出勤時に「直行」、退勤時に「直帰」と別々に記録したい）
[Answer]打刻レコードに1つずつ紐づく

### 後から編集時の UI

[Question] 後からメモを編集する場合、どの画面から操作しますか？（勤怠履歴一覧の編集ボタン / 日次詳細画面 / 当日ステータス画面）
[Answer]勤怠履歴一覧の編集画面

### 打刻フローとメモ入力の関係

[Question] 打刻ボタン押下後にメモ入力が表示されるとのことですが、フローはどちらですか？
A) ボタン押下 → 打刻が即確定 → その後メモ入力画面が出る（メモはスキップ可能）
B) ボタン押下 → メモ入力画面が出る → 「確定」ボタンで打刻＋メモが同時に保存される
[Answer]B

### カテゴリと打刻種別の関係

[Question] カテゴリの中に「直行」（出勤向き）や「直帰」（退勤向き）がありますが、出勤/退勤に応じて選べるカテゴリを制限しますか？それともすべてのカテゴリを常に選択可能にしますか？
[Answer]出勤／退勤に応じて選べるようにする

### 「その他」選択時の自由テキスト

[Question] カテゴリで「その他」を選んだ場合、自由テキストの入力は必須にしますか？（理由の説明を強制する）
[Answer]必須にする

### メモ任意とフロー B の整合

[Question] メモは「任意」で、フローは「B（ボタン押下→メモ入力→確定で同時保存）」とのことですが、メモを書かずに打刻したい場合のフローはどちらですか？
A) ダイアログに「メモなしで打刻」ボタン（スキップ）を用意する
B) ダイアログ自体を出さない選択肢（例: 長押しでメモなし即打刻）を用意する
C) カテゴリ未選択・テキスト空欄のまま「確定」を押せばメモなし打刻になる
[Answer]A

### カテゴリの出勤/退勤 振り分け

[Question] 以下の振り分けで合っていますか？修正があれば教えてください。
- 出勤時: 直行、電車遅延、在宅、障害対応、その他
- 退勤時: 直帰、早退、外出、障害対応、その他
[Answer]退勤時にも在宅をいれてほしい

## 確定仕様まとめ

| 項目 | 仕様 |
|------|------|
| 対象タイミング | 出勤・退勤の両方 |
| 必須/任意 | 任意（「メモなしで打刻」スキップボタンあり） |
| 文字数上限 | 200文字 |
| 入力形式 | カテゴリ（単一選択・必須）＋ 自由テキスト |
| 紐付き単位 | 打刻レコード（出勤/退勤）ごとに1メモ |
| 編集・削除 | いつでも可能（勤怠履歴一覧の編集画面から） |
| 閲覧範囲 | 本人 ＋ 上長 |
| 表示箇所 | 打刻直後の確認・勤怠履歴一覧・日次詳細（すべて） |
| UI フロー | 打刻ボタン押下 → ダイアログ表示 → 「確定」で打刻＋メモ同時保存 / 「メモなしで打刻」でスキップ |
| 「その他」時 | 自由テキスト入力必須 |
